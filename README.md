# PromptOS — AI Prompt Marketplace

Browse, share, and sell AI prompts for ChatGPT, Claude, Midjourney, Flux, and more. Test prompts across multiple models, optimize them, and compete in battles.

## Features

- **Marketplace** — Browse hundreds of prompts across 12+ categories
- **Lab** — Test any prompt across GPT-4o, Claude 3.5, and Gemini 1.5 side-by-side
- **Optimizer** — Improve, rewrite, or convert prompts for different AI models
- **Battles** — Vote for the best prompts in head-to-head matchups
- **Community** — Discuss prompts, share tips, and connect with creators
- **Creator Dashboard** — Upload, sell, and track your prompts

## Tech Stack

- **Frontend:** TanStack Start, React, Tailwind CSS, Framer Motion
- **Backend:** Supabase (PostgreSQL, Auth, Storage)
- **AI:** OpenRouter (GPT-4o, Claude 3.5, Gemini 1.5)
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```
